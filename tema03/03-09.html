Servidores Web

2.2. ¿Cuáles son las tecnologías asociadas a las aplicaciones web?

Las aplicaciones web modernas emplean páginas dinámicas que se procesan de dos formas principales: en el servidor (antes de enviarse al navegador) o en el cliente (dentro del navegador del usuario). Muchas aplicaciones actuales utilizan una combinación de ambos enfoques. Veamos las tecnologías más relevantes en cada contexto:

Tecnologías del lado del servidor (Backend)

Estas tecnologías se ejecutan en el servidor web y generan contenido dinámico antes de enviarlo al navegador del usuario.

CGI (Common Gateway Interface — Interfaz Común de Pasarela)

  • Uno de los estándares más antiguos para aplicaciones web dinámicas, desarrollado en los años 90.
  • Permite que un servidor web ejecute programas externos y devuelva su salida al cliente.
  • Históricamente se usaba para formularios web, contadores de visitas, y funcionalidades básicas.
  • Generalmente implementado en lenguajes como Perl, C, o scripts de shell.
  • En 2025, CGI está prácticamente obsoleto, reemplazado por tecnologías más modernas y eficientes.

PHP (Hypertext Preprocessor)

  • Uno de los lenguajes más populares para desarrollo web del lado del servidor.
  • Código abierto y multiplataforma.
  • Fácil de aprender e integrar con HTML.
  • Excelente soporte para bases de datos (MySQL, PostgreSQL, MongoDB).
  • Gran ecosistema de frameworks (Laravel, Symfony, WordPress).
  • En 2025, PHP sigue siendo muy utilizado, especialmente en su versión 8.x, que ha introducido mejoras significativas de rendimiento y características modernas del lenguaje.
  • Utilizado por sitios importantes como Facebook, Wikipedia, y WordPress.

ASP.NET

  • Desarrollado por Microsoft como evolución de ASP clásico.
  • Framework robusto para desarrollo de aplicaciones web empresariales.
  • Soporte para múltiples lenguajes (C#, VB.NET, F#).
  • En 2025, ASP.NET Core es multiplataforma (Windows, Linux, macOS) y de código abierto.
  • Excelente integración con el ecosistema Microsoft.
  • Alto rendimiento y escalabilidad.
  • Muy utilizado en entornos corporativos y empresariales.

Java (Ecosistema Java EE / Jakarta EE)

  • Plataforma madura y robusta para aplicaciones web empresariales.
  • Tecnologías clave:
    • Servlets: clases Java que manejan peticiones HTTP
    • JSP (JavaServer Pages): similar a PHP pero con Java
    • JSF (JavaServer Faces): framework de componentes UI
    • Spring Framework: el framework Java más popular en 2025
  • Muy utilizado en grandes corporaciones y aplicaciones bancarias/financieras.
  • Excelente para aplicaciones que requieren alta seguridad y fiabilidad.

Python

  • Lenguaje versátil y de sintaxis clara.
  • Frameworks web populares:
    • Django: framework completo y robusto, con ORM incluido
    • Flask: microframework ligero y flexible
    • FastAPI: framework moderno para APIs, muy rápido
  • Muy popular en 2025 para desarrollo rápido de prototipos y aplicaciones.
  • Excelente para integración con IA y machine learning.
  • Usado por sitios como Instagram, Pinterest, y Spotify.

Ruby

  • Conocido especialmente por Ruby on Rails, un framework que revolucionó el desarrollo web.
  • Énfasis en convención sobre configuración.
  • Desarrollo rápido de aplicaciones.
  • Aunque su popularidad ha disminuido respecto a su pico en 2010-2015, sigue siendo utilizado por empresas establecidas.

Node.js / JavaScript del lado del servidor

  • Permite ejecutar JavaScript en el servidor, no solo en el navegador.
  • Modelo de E/S no bloqueante, ideal para aplicaciones en tiempo real.
  • Frameworks populares:
    • Express.js: minimalista y flexible
    • Nest.js: inspirado en Angular, muy estructurado
    • Fastify: enfocado en rendimiento
  • NPM: el mayor ecosistema de paquetes del mundo.
  • En 2025, muy popular para desarrollo "full-stack" (mismo lenguaje en cliente y servidor).
  • Ideal para aplicaciones que requieren muchas conexiones simultáneas (chat, juegos online, streaming).

Go (Golang)

  • Lenguaje desarrollado por Google.
  • Excelente rendimiento y eficiencia.
  • Concurrencia nativa.
  • Cada vez más popular para microservicios y APIs de alto rendimiento.
  • Sintaxis simple y clara.

Tecnologías del lado del cliente (Frontend)

Estas tecnologías se ejecutan en el navegador del usuario.

HTML (HyperText Markup Language)

  • El lenguaje fundamental de la web, utilizado para estructurar el contenido.
  • En 2025, HTML5 es el estándar, incorporando:
    • Elementos semánticos (article, nav, section)
    • Soporte multimedia nativo (video, audio)
    • APIs para aplicaciones avanzadas (geolocalización, almacenamiento local, canvas para gráficos)
  • Base sobre la que se construyen todas las páginas web.

CSS (Cascading Style Sheets)

  • Define la presentación visual de las páginas web.
  • Separa contenido (HTML) de diseño (CSS).
  • CSS3 incluye:
    • Animaciones y transiciones
    • Diseño responsive (media queries)
    • Flexbox y grid para layouts complejos
    • Variables CSS
    • Filtros y efectos visuales avanzados
  • Preprocesadores populares en 2025: Sass, Less, PostCSS.
  • Frameworks CSS: Tailwind CSS, Bootstrap 5, Material Design.

JavaScript

  • El lenguaje de programación de la web, esencial para interactividad.
  • En 2025, JavaScript moderno (ES2024+) incluye:
    • async/await para programación asíncrona
    • Módulos ES6
    • Características de programación funcional
    • Mejoras constantes del lenguaje
  • Frameworks y bibliotecas principales:
    • React (de Meta/Facebook, muy popular para UIs complejas)
    • Vue.js (progresivo y fácil de aprender)
    • Angular (de Google, framework completo)
    • Svelte (compilador que genera código muy eficiente)
  • TypeScript: superset de JavaScript que añade tipos estáticos, muy popular en 2025 para proyectos grandes.
  • JavaScript permite:
    • Manipulación del DOM (modificar la página dinámicamente)
    • Peticiones asíncronas (AJAX, Fetch API)
    • Validación de formularios
    • Animaciones e interacciones complejas
    • Aplicaciones de una sola página (SPA)

WebAssembly (Wasm)

  • Tecnología relativamente nueva que permite ejecutar código de alto rendimiento en el navegador.
  • Puedes escribir en lenguajes como C, C++, Rust, y compilar a WebAssembly.
  • Rendimiento cercano al nativo.
  • Útil para:
    • Juegos en el navegador
    • Edición de vídeo/audio online
    • Aplicaciones CAD
    • Cualquier aplicación que requiera cálculos intensivos
  • En 2025, cada vez más adoptado para aplicaciones exigentes.

Progressive Web Apps (PWAs)

  • Aplicaciones web que funcionan como aplicaciones nativas.
  • Características:
    • Funcionan offline
    • Se pueden instalar en el dispositivo
    • Reciben notificaciones push
    • Acceso a hardware del dispositivo (cámara, GPS)
  • Basadas en tecnologías web estándar (HTML, CSS, JavaScript).
  • Service Workers permiten cacheo avanzado y funcionamiento offline.
  • En 2025, ampliamente adoptadas como alternativa a aplicaciones nativas.

Tecnologías obsoletas o en desuso

  • Java Applets: prácticamente extintos debido a problemas de seguridad y rendimiento.
  • Adobe Flash: oficialmente descontinuado en 2020, completamente obsoleto en 2025.
  • VBScript: solo funcionaba en Internet Explorer, que fue reemplazado por Edge. Ya no se usa.
  • Silverlight: tecnología de Microsoft descontinuada, reemplazada por HTML5.
